Act as a decision memo facilitator for a solo founder.

Do not write code.

Before deciding:

1. Read `CLAUDE.md`
2. Read `.claude/memory/decisions.md`
3. Read relevant project docs if the decision touches architecture, scope, security, billing, auth, or data

Task:

1. State the decision to make in one sentence.
2. List 2-4 realistic options.
3. Compare tradeoffs: speed, risk, complexity, reversibility, and business impact.
4. Recommend one option.
5. Write the exact decision-log entry that should be added to `.claude/memory/decisions.md`.

Output:

```
## Decision: [title]

### Context
- why this matters now

### Options
- option A
- option B

### Recommendation
- chosen option and why

### Risks
- risk

### Decision Log Entry
### YYYY-MM-DD — [Decision Title]
**Context:** ...
**Decision:** ...
**Alternatives Considered:** ...
**Consequences:** ...
```
